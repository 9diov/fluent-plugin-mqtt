# Fluent::Plugin::Mqtt

Fluent plugin for MQTT protocol

## Installation

Add this line to your application's Gemfile:

    gem 'fluent-plugin-mqtt'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install fluent-plugin-mqtt

## Usage

This client works as ONLY MQTT client.
